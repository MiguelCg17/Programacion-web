¿COMO HACER FLOTAR CAJAS?

Caja 1

Para hacer que las cajas floten en HTML, necesitas utilizar CSS con la propiedad float. Esto se usa comúnmente para alinear elementos como cajas o divs a la izquierda o derecha dentro de su contenedor.

Caja 2

La propiedad float: left; hace que la caja se alinee a la izquierda. La propiedad float: right; hace que la caja se alinee a la derecha. La clase contenedor con overflow: auto; asegura que el contenedor se ajuste correctamente al contenido flotante, evitando problemas de colapso de contenedor.
